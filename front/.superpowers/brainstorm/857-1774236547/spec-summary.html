📐 布局结构
桌面端：三栏布局 — 左 (任务树 25%) / 中 (详情 45%) / 右 (AI 聊天 30%)
移动端：根任务网格 → 点击全屏推开 → 面包屑导航 + 子任务列表
💬 AI 聊天
上下文绑定：支持针对根任务或任意子任务聊天
移动端：底部抽屉半屏 → 上滑全屏
智能修改：AI 可修改任务任意元素 (标题/描述/步骤/负责人/质量门/依赖/状态)
修改预览：红绿对比显示删除/新增，用户确认后应用
📋 任务功能
编号规则：根任务无编号，子任务 1,2,3 → 1.1,1.2 → 1.1.1,1.1.2
依赖关系：仅允许依赖同级或直接父级，禁止跨层跨分支
拖拽排序：支持子任务拖拽重排，自动重算编号
状态管理：待办 / 进行中 / 阻塞 / 已完成
质量门：可配置 JSON Schema 验证
👆 移动端手势
点击：进入下一级任务列表
左滑：删除/阻塞操作
右滑：查看详情
长按：AI 分解菜单
抽屉上滑：半屏聊天 → 全屏聊天
🎨 视觉风格
双主题：深色 (深空灰 + 电光蓝) / 浅色 (纯白 + 科技蓝)，支持一键切换
组件：渐变顶边、左侧高亮、发光阴影
字体：系统字体 + JetBrains Mono 代码字体
图标：Lucide 图标库
⚛️ 技术栈
框架：React 18 + Vite
状态管理：Zustand / Redux Toolkit
样式：Tailwind CSS + CSS Variables
Markdown：react-markdown + TipTap
拖拽：dnd-kit / react-beautiful-dnd
Mock 数据：30 个根任务，深度 4 层，MSW 拦截
